این کتاب یکی از کتاب های خوب و همچنین معروف برای یادگیری برنامه نویسی است. چنانکه از اسم کتاب مشخص است و در مقدمۀ آن هم آمده، این کتاب از زبان جاوا استفاده می کند و به وسیلۀ آن به شما آموزش می دهد که چگونه مانند یک دانشمند کامپیوتر فکر کنید. نویسنده های این کتاب، کتاب های دیگری هم در زمینۀ برنامه نویسی نوشته اند که احتمالا Think Python معروف ترین آنهاست. تمامی کتاب های این دو نویسنده از جمله نسخۀ اصلی این کتاب را می توانید از وبسایت آنها به صورت رایگان دانلود کنید.
